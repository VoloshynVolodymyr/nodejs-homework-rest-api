## Usage

```python
import foobar

# Return all contacts
get .../api/contacts

# Return contact by "id"
get .../api/contacts/:id

# Add a contact to the contact list
post .../api/contacts

# Delete contact by "id"
delete .../api/contacts/:id

# Update contact details by "id"
put .../api/contacts/:id

# Change the "favorite" field for a contact by "id"
patch .../api/contacts/:id/favorite
```
